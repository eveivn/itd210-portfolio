# Portfolio Website — Evelyn Garcia

## About This Project
This portfolio website showcases artwork created by an illustration team that focuses on publishing web comics and digital art. The main objective is to present the artwork following responsive web design principles that can be accessible on any device. 

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- JavaScript
- Tailwind CSS
